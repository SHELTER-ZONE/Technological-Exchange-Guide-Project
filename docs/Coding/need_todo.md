### 附上對應完整的程式碼與圖片
請善於使用CodeBlock，將你的程式碼/錯誤碼額外框起來  
這樣一來版面更整潔，也更方便他人複製你的Code下來除錯  
#### 範例:
> **可以請教為甚麼這無法運作嗎?**  
> ```
> #include <iostream>
> int main()
> {
>     std::cout << "Hello World"
>     return 0;
> }
> ```
>> **請問這個錯誤是甚麼意思?**  
>> ```HelloWorld.cpp:4:31: error: expected ';' before 'return' std::cout << "Hello World"```
