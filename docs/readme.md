# 介紹
## 關於本文件 [Developer Stage]
本文件為 SHELTER ZONE 所協作編寫「如何提問技術問題」之指南文件。

主要為提供於本伺服器內 `新進人成員`、`各領域初學者`協助其對相關技術問題之提問技巧指南與注意事項，希望藉此達到更有效率的答覆。
